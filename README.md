## ERP
```mysql
ERP
전사적 자원 관리
원가관리
자재관리
영업관리
인사관리
고정자산

기업의 회계부터 인사 및 제조관리 등 일상 업무 활동을 통합관리하는
시스템 및 소프트웨어 패키지

AS-IS : 부서별로 시스템을 따로관리. 정보가 공유가안됨
TO-BE : 데이터 중복제거, 한 부서에서 데이터를 입력하면, 전 부서가 정보를 활용가능

ERP는 생산/구매 영역을 넘어, 인사,회계 등 MIS(경영정보시스템) 기능까지 탑재

기업 외부까지 관리하기위해 ERP-2가 됨
CRM,SCM까지 통합하는 개념으로

```
